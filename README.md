# ПРОЕКТ GIT INFO
Этот **проект** предназначен для объяснения работы с **git** и **github**.
____
## GIT и GITHUB. Что это такое и для чего?
GIT – это программа контроля версий проекта.
GITHUB – это платформа для хранения проектов и совместной работы над ними с использованием GIT.
____
## С чего начать? (Для пользователей *Windows*)
1.	Открываем программу **Git Bash**
2.	Создаем в необходимом нам месте папку будущего проекта (mkdir *название проекта*)
3.	Переходим в созданную папку проекта (cd *название проекта*)
4.	Ввести команду **git init** (в подпапке .git хранится история изменений)
5.	Подготовить файлы к сохранению **git add .** (не забудь точку)
6.	Выполнить коммит **git commit –m 'комментарий к коммиту’**
Команда **git log**показывает выводит историю коммитов.
____
###Часть GITHUB

7.	Создаем репозиторий проекта на GITHUB
8.	Привязываем локальный и удаленный репозиторий через команду:
**git remote add origin *git@github.com:%ИМЯ_АККАУНТА%/first-project.git* **
9.	Проверяем командой, что все работает: **git remote –v**
10.	Отправить первое изменение на удаленный репозиторий ** git push -u origin main**
Далее отправляем изменения на сервер через команду **git push**
